# A-Game-About-Crates-Save-Editor
-just a quick save editor i made for the vrc world a game about crates
-the worlds save system requires you to input your vrc username to decrypt the save if you dont trust this then dont use the script numbers in a random world arent worth running things you dont trust
-be sure to have the { } around your save data when you try and re encrypt 
-and its easier to just save the data as a file then use the file path for re encryption
